## Learnings
- Single-row subqueries return one value, multi-row subqueries need IN
- Correlated subqueries use the outer query value in the inner query
- EXISTS and NOT EXISTS are great for checking presence or absence of rows
- `manager_id` is a self-reference in the employees table
- Always use IS NULL instead of `= NULL`
- Using DISTINCT helps avoid duplicates in EXISTS queries
