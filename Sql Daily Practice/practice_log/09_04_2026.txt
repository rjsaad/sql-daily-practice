##Learnings
ROW_NUMBER, RANK, DENSE_RANK handle ties differently
FIRST_VALUE gives value but does not filter rows
LAST_VALUE and NTH_VALUE need proper window frame
Default window frame can cause wrong results
LEAD and LAG require ORDER BY
LAG returns NULL for first row
SUM OVER is used for cumulative sum
ORDER BY changes meaning (amount vs date)
Correlated subquery uses outer query column
Subquery with GROUP BY returns multiple rows
