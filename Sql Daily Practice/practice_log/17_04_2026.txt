## Learnings

- COUNT(*) counts all rows including NULL-containing rows
- COUNT(column) ignores NULL values
- NULL cannot be compared using = operator
- LEFT JOIN keeps all rows from left table
- JOIN can increase row count (one-to-many expansion)
- GROUP BY changes calculation scope
- HAVING filters aggregated results, WHERE does not
- JOIN duplication can lead to incorrect aggregates
