# SQL Daily Practice Repository

This repository is dedicated to **daily SQL practice**, covering everything from **basic queries** to **advanced SQL concepts**. It is designed to help you **learn, track, and showcase your SQL skills** step by step.

---

## 📅 How to Use

1. **Setup the database**  
   Open `dataset/schema.sql` to create tables and insert sample data.

2. **Attempt daily exercises**  
   Read the questions in `practice-log/YYYY-MM-DD.md` and try solving them.

3. **Write your solutions**  
   Save your SQL queries in `solutions/YYYY-MM-DD.sql`.

4. **Review and learn**  
   Compare your solutions with correct queries, note mistakes, and write your learnings in the daily log.

5. **Commit and push daily**  
   Track your progress with Git and upload updates to GitHub.

---

## ✅ Topics Covered

- **Basic Queries:** SELECT, WHERE, ORDER BY, LIMIT, DISTINCT, string & numeric functions  
- **Aggregates:** COUNT, SUM, AVG, MAX, MIN, GROUP BY, HAVING  
- **Joins:** INNER JOIN, LEFT/RIGHT JOIN, FULL OUTER JOIN, self-joins  
- **Subqueries:** Single-row, multi-row, correlated subqueries, EXISTS / NOT EXISTS  
- **Set Operations:** UNION, INTERSECT, EXCEPT  
- **Advanced SQL:** Window functions (ROW_NUMBER, RANK, etc.), CTEs (WITH clause), CASE statements  
- **Indexes & Constraints:** Primary keys, foreign keys, UNIQUE, NOT NULL
